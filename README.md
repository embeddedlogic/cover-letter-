# Cover Letter Generator
A Flask web application that generates customizable cover letters.

## Features
- User input form
- Generates personalized cover letters

## Technologies
- Python
- Flask
- HTML
- CSS
## How to Run

1. Clone the repository

git clone https://github.com/embeddedlogic/cover-letter-generator.git

2. Go to the project folder

cd cover-letter-generator

3. Install dependencies

pip install flask

4. Run the Flask application

python app.py

5. Open the website in your browser

http://127.0.0.1:5000
